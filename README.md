# diffentropy

# 操作步骤
1. 克隆仓库到本地
```
git clone https://github.com/cyruscyliu/diffentropy.git
```
2. 链接远程仓库
```
git remote add origin git@github.com:cyruscyliu/diffentropy.git
```
3. 提交修改
```
git add *
git commit -m 'description'
git push -u origin master
```
4. 拉取更新
```
git pull origin master:master
```
